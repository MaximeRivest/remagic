# Walkthrough photos

Drop the developer-mode walkthrough photos here, named exactly as below (they're
referenced by `docs/DEVELOPER-MODE.md`). JPG or PNG both fine.

| File | The shot to take |
|------|------------------|
| `01-open-settings.jpg` | The home screen with the account/gear tapped, or the Settings screen open |
| `02-general-software.jpg` | Settings → General, showing the **Software** entry |
| `03-advanced-developer-mode.jpg` | The **Advanced** menu showing **Developer mode** |
| `04-enable-warning.jpg` | The warning screen about the factory reset (the important one) |
| `05-confirm-power-button.jpg` | The "press the power button twice to confirm" prompt |
| `06-ssh-password.jpg` | Settings → Help → Copyrights and licenses, the SSH password/address section |
| `07-ssh-connected.jpg` | Your laptop terminal after a successful `ssh root@10.11.99.1` |

Tips: straight-on, good light, no glare on the e-ink; frame the whole screen
with a little bezel so people can orient. A short screen recording split into
stills works too.
