From stock to spellbook

One command. Then no computer.

01 — PREPARE

Enable developer mode

Follow our visual walkthrough. reMarkable requires a factory reset, so sync your notebooks first.

02 — CONNECT

Run the installer

Plug in over USB and run one command. remagic installs the launcher, persistence, and Store.

03 — EXPLORE

Open the Store

From then on, discover and install checksum-verified apps right on the tablet.
